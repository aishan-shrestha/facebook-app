# This app is just a demo sample for know how the app is used. May requires some tweak further
Step 1: Download these files
Step 2: Run composer to download Facebook SDK, i.e. composer update. And change path as needed   require_once __DIR__ . '/vendor/autoload.php'; 
Step 3: Create a folder "cover_pics" and add few images with names (see on index.php)

Note
- The Facebook SDK requires PHP version 5.4 or higher.'
- Requires to submit APP approval to access "public action" and "user photo" permission on your FB App account
